## ✨ 核心功能

### 🌍 全球视频下载支持

- **支持 1000+ 网站** - 通过 yt-dlp 引擎下载全球几乎任何网站的视频
- **智能平台检测** - 自动检测视频平台并优化下载参数
- **本地化界面** - 支持多语言界面

### 🎨 一流的用户体验

- **现代化设计** - 简洁美观的界面
- **实时进度** - 详细的下载进度跟踪和状态管理
- **主题切换** - 支持系统/浅色/深色主题，舒适观看

## 🌐 支持的网站

CC-Download 通过 yt-dlp 支持数百个视频和音频平台。

> **💡 提示：** CC-Download 底层使用 [yt-dlp](https://github.com/yt-dlp/yt-dlp) 引擎，支持 1000+ 网站。完整列表请访问 [yt-dlp 支持的网站文档](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)。
